🤖 UR10e Automated Clipper Project

This project focuses on the design, fabrication, and programming of a custom robotic end-effector for a Universal Robots UR10e. The goal is to create a reliable system for automated clipping/trimming tasks using a standard electric clipper.
📌 Project Overview

As part of my Electrical Engineering studies at the University of Oulu, I am developing this mount to bridge the gap between industrial robotics and everyday tools. This repository serves as a complete documentation of the mechanical CAD, electrical integration, and software control.
📂 Project Modules

The project is divided into three main sections. Click the links below to view the specific documentation for each:

    🏗️ Mechanical Design: Fusion 360 models, 3D printing parameters (PETG), and assembly hardware.

    ⚡ Electronics: Power management for the clipper, relay/transistor switching, and wiring diagrams.

    💻 Software & Control: URScript programs, safety planes, and path planning for the robot wrist.

🛠 Work in Progress (Roadmap)

The project is currently in the active development phase. Below is the status of the upcoming tasks:

    [x] Phase 1: Mechanical Design (Finished CAD of the main mount, flange interface, and C-clamp).

    [ ] Phase 2: Initial Prototyping (3D printing the first iteration at Fab Lab Oulu for fitment testing).

    [ ] Phase 3: Electrical Actuation (Wiring the clipper to be triggered via the UR10e Tool I/O port).

    [ ] Phase 4: Software Implementation (Writing the URScript logic to handle the clipping motion).

    [ ] Phase 5: Safety Testing (Validating force limits and emergency stop behavior).

🚀 Getting Started
